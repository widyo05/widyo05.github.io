My Biography

Saya lahir di Salatiga dan besar di Solo. Saya menamatkan pendidikan SD sampai SMA di kota Solo dari tahun 1999 - 2005. Kemudian saya kuliah di Universitas Duta Wacana Yogyakarta jurusan Teknik Informatika dari tahun 2005 - 2011. Saya tertarik belajar IT karena belajar IT dapat mengasah logika berpikir dan melalui IT, saya bisa berpartisipasi untuk memajukan SDM Indonesia di bidang Teknologi. Selama berkecimpung di IT, saya pernah mengikuti beberapa pelatihan antaralain :
1. Cisco Network Fundamentals
2. Cisco Routing Protocols
3. Pemrograman Mobile berbasis Kotlin
